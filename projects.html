Scientific Codebase RAG

Python, notebooks, code retrieval, scientific computing, RAG evaluation

An active research repository for retrieval-augmented generation over scientific software. The project studies why scientific code breaks generic RAG assumptions: users ask in mathematical language, code answers in APIs, and useful context often lives across functions, docstrings, and cross-references.

Current experiments compare AST-aware chunking, code-aware embeddings, BM25 plus dense search, reciprocal rank fusion, and LLM-generated summaries. The roadmap moves toward Qdrant hybrid search, cross-encoder reranking, richer math metadata, and execution-grounded evaluation for coding agents.

Insider Threat Detection System

Python, FastAPI, XGBoost, Isolation Forest, Vanilla JavaScript

A Master of Software Development capstone project that detects malicious insider behavior in enterprise activity logs. The pipeline ingests raw CERT dataset logs, engineers daily behavioral features, simulates time day by day, and fuses four detector outputs into analyst-facing alerts.

The project emphasizes measured trust: supervised detection, forecast warning, rules, anomaly detection, ensemble weighting, and a dashboard that explains why a user escalated instead of presenting a black-box score.

TLSlite

Java, TCP sockets, Diffie-Hellman, HKDF, AES-CBC, HMAC

A simplified TLS-like protocol implementation built for learning the mechanics of authenticated key exchange and encrypted messaging. The client and server perform mutual certificate authentication, ephemeral Diffie-Hellman key exchange, HKDF-based session key derivation, transcript HMAC checks, and authenticated encrypted messages over TCP.

The README is explicit about the project's limits, including static IV reuse, missing DH public-key validation, timing-unsafe HMAC comparison, and no production-grade certificate chain validation. That honesty is part of the point: security work has to name what is not protected.

DNS Resolver

Java, UDP networking, RFC 1035 parsing, TTL-aware caching

A local DNS forwarding resolver that listens for UDP queries, parses RFC 1035 wire-format messages, handles compressed domain-name pointers, checks an in-memory TTL-aware cache, and forwards cache misses to Google Public DNS.

The implementation covers DNS packet construction, bit-level header flag extraction, custom cache-key equality, record expiration, and response generation. It also documents concrete limitations, including single-threaded handling, no TCP fallback, no DNSSEC, and volatile cache state.

Cosmic Cat Combat

C++, Qt, QGraphicsScene, Qt Multimedia, game-state logic

A Qt/C++ fixed shooter with sprite-based rendering, focused keyboard input, timer-driven projectile movement, collision detection, local profiles, scoreboard navigation, and multimedia cues.

The game is also a study in state management. The boss character moves through peek, retreat, monologue, attack, victory, and defeat phases; the player loop coordinates health, projectiles, replay flow, and screen transitions through a stacked-widget Qt interface.
